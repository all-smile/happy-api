#### 安装教程

1.  go get 安装依赖
2.  go mod vendor 安装依赖包到 vendor 项目目录

#### 使用说明

1.  本地开发： go run main.go 或者 bee run
2.  打包 bee pack 生成happy-api.tar.gz
3.  本地build生成.exe文件： go build .
